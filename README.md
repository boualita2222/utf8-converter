# UTF-8 Converter Pro

Convertisseur UTF-8 complet en Python 3.14.2

## Applications
- Interface graphique Tkinter
- API REST Flask
- Site web Django 6.0
- Application .exe PyInstaller

## Fonctionnalites
- Encodage / Decodage UTF-8 (1-4 octets)
- Analyse Unicode complete
- Base de donnees SQLite
- 37 tests unitaires
- Rapports PDF, Excel, Word

## Installation
pip install django flask requests colorama reportlab openpyxl python-docx

## Lancement
python utf8_gui.py
python utf8_api.py
python manage.py runserver
